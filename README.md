# Angular-Commands

## Instalación:
```
npm install -g @angular/cli@latest
```
## Crear proyecto:
```
ng new
```
## Abrir Servidor:
```
ng serve
```
## Crear Componente nuevo:
```
ng g component (Lugar Donde lo quieras crear)/(Nombre Carpeta a crear)
```
## Ver versión de Angular:
```
ng version
```
